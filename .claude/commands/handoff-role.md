Ты завершаешь работу из-за заполнения контекста и передаешь дела преемнику в той же роли.

**ВАЖНО:** Думай тщательно о том, что критично передать. Из-за заполненного контекста ты можешь упустить важное.

## Шаг 1: Создай файл передачи дел

**Имя файла зависит от твоей роли:**
- Architect → `00_docs/_architect_handoff.md`
- Analyst → `01_tasks/NNN_название/_analyst_handoff.md` (где NNN - текущая задача)
- Developer → `01_tasks/NNN_название/_developer_handoff.md`
- Reviewer → `01_tasks/NNN_название/_reviewer_handoff.md`

**Структура файла передачи:**

```markdown
# Передача дел: [Твоя роль]

**Дата:** YYYY-MM-DD
**Задача:** [Номер и название, если применимо]

## Что сделано
- [Конкретное действие 1 с результатом]
- [Конкретное действие 2 с результатом]
- [...]

## Текущий статус
[На каком этапе сейчас. Одно предложение.]

## Ключевые решения
- [Решение 1: что решили и почему]
- [Решение 2: что решили и почему]

## Что важно знать преемнику
- [Критичная деталь 1 - что может быть неочевидно]
- [Критичная деталь 2]
- [Риски или подводные камни]

## Следующие шаги
1. [Конкретное действие 1]
2. [Конкретное действие 2]
3. [...]

## Файлы для чтения
- `[путь/к/файлу1]` - [зачем читать]
- `[путь/к/файлу2]` - [зачем читать]
```

**Требования к файлу:**
- Будь конкретен: не "продолжить анализ", а "добавить раздел 3.4 про обработку ошибок"
- Укажи только критичное - не дублируй очевидное
- Явно перечисли файлы которые нужно прочитать

## Шаг 2: Сгенерируй промпт для преемника

После создания файла передачи создай промпт для запуска преемника:

```
Ты — агент [Твоя роль] (см. .agents/[твоя роль].md).

Прочитай:
- .agents/[твоя роль].md
- [файл передачи дел из Шага 1]
- [файл 1 из "Файлы для чтения"]
- [файл 2 из "Файлы для чтения"]
- [...]

Контекст: Ты продолжаешь работу предыдущего [Роль]. Изучи переданные дела и продолжай с того места.

После изучения контекста сообщи что готов продолжать работу.
```

**Положи промпт в markdown блок (```) для удобного копирования.**

## Важные напоминания

- Не включай в промпт содержимое файлов - только пути
- Перечисли конкретные файлы, не пиши "релевантные ADR" или "текущий код"
- Промпт должен быть 7-10 строк максимум
- В файле передачи фокусируйся на неочевидном, а не на том что и так в файлах
